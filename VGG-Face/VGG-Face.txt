VGG Face Descriptor

Details:
The dataset consists of 2,622 identities. Each identity has an associated
text file containing URLs for images and corresponding face detections. 
Models pretrained using this data can be found at VGG Face Descriptor(http://www.robots.ox.ac.uk/~vgg/software/vgg_face/) webpage.
